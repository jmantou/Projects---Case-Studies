Understanding the Basics of HTML

Basics of HTML, XHTML and HTML5

HTML is a basic but terrific technology for a number of reasons:

  • It uses plain text, and thus it has a universal format that can be used with any computer. That means that any plain computer with a simple text editor can read and write it.
  • It describes what web documents mean instead of what a document should look like. HTML purpose is to describe the meaning of various page elements. When you figure out how to use it properly you can understand the advantages of the above statement.
  • It's easy to write. Of course not all web pages as simple as our first example, but you can easily learn to write HTML by learning the basic tags and elements which are pretty straightforward.
  • It's free. The fact that this technology is available to anyone for free is a huge advantage.

Learning the Tags

The key to writing HTML code is the special text you see in a web page inside the angle braces (<>). These special elements are called tags. They are not meant to be displayed in a page but offer instructions to the browser about the meaning of the page. They can be embedded into each other to further help with the organization of the page.
