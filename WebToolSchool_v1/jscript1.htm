Introduction to JavaScript

Basics of the JavaScript Language

Web pages are defined by the HTML language and are styled with CSS. But in order to give them life and interactivity you need a suitable programming language. Programming with JavaScript is what makes pages interact with users.

JavaScript is a programming language developed specifically this purpose, giving life to web pages while a user interacts with them. The language is implemented in all the major browsers for many years and it's a major feature of them in order to properly display web pages.

Working with JavaScript

You should know some basic things about JavaScript:

  • Scripting Language.JavaScript is not a compiled language which means that it does not require any special steps to be made to produce an executable file.
  • Not Java. It has nothing to do with the Java language except some syntax that is common in languages of the C family.
  • Fully Featured Language.It has all the hallmarks of a complete computer language although many features have a more weirder twist than in other languages.

Example

<script>
     document.write("Hello World!");
</script>
