Creating a Basic XHTML Page

Introduction to HTML

We will start by building a simple HTML page and learning the basic elements of web pages constructing an example. HTMl is a more older standard than XHTML and HTML5, but is a bit more easier for beginners to understand and learn the basic steps.

Basic Steps

In order to build a basic page we don't need any expensive and complicated software nor a powerful computer. The basic tools are available in any ordinary Windows or Linux machine.

OK then! Let's fire the computer and build our first page!

Example

Open a simple text editor and type the following code:

<html>
<head>
<title>My first Html Page</title>
</head>
<body>
<h1>The Heading Title</h1>
<p>This is my first, surely very simple web page!</p>
<p>It is so cool!</p>
</body>
</html>

A Web Browser is a program that you'll use to view html pages. There are a number of very popular web browsers, but for start you can use whichever you like. Because you new file has not been uploaded yet to a public server you can simply locate the page file you just made in your file system and double click on it to open it in your default browser for viewing.
